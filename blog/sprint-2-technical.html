Inline versus inline-block

20 April 2021

Here's the difference between inline and inline-block.
As you can see this inline-block and its padding
are pushing the rest of the text around.
On the other hand, this inline element with exactly the same CSS rules doesn't do that.
The only rule difference between these two is display:inline versus display: inline-block.

Both these elements have the following rules: 

padding: 20px
height: 100px;
width: 200px;
but inline elements totally ignore width & height, and they ignore the top and bottom margin & padding values.
Note in the above example that the inline element still respects the padding rule, and has a 20px gap to both its left and right.
